This code is used to plot charge density images from the data extracted using getting_charge_density codes. Data from getting_charge_density code is saved in the form of excel sheet, in the format explained in that code. This code reads the data from excel shet and plot a png image in the same folder. This code requires three inputs, first the name of the excel file, second is structure id (can be anything, user can input 1 as well), and third is the value of max intensity. Users have to set the value of max intensity depending upon the system.
